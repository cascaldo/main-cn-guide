# main-cn-guide

本仓库用于归档和发布多个独立的 HTML 页面。这些页面不针对任何单个域名或具体网站，而是作为一个通用文档与页面资源的集合地。

## 项目简介

该仓库的主要目的是存放和展示若干自包含的 HTML 文件，每个文件均可独立运行，无需额外依赖。内容涵盖各类说明、指引或静态页面，适合需要快速访问和存档的场景。

## 目录说明

- **根目录**：存放主要的 HTML 页面文件，每个文件以有意义的名称命名，便于识别。
- **assets/**（如存在）：包含页面所需的静态资源，如样式表、图片或脚本文件。
- **archive/**（如存在）：存放历史版本或不再活跃的页面，仅作归档用途。

## 页面归档说明

每个 HTML 页面均为独立单元，不依赖外部服务器或数据库。页面内容可能随时间更新，但历史版本会保留在 `archive/` 目录中。用户可直接通过仓库的 Pages 功能或本地打开方式访问这些页面。

## 维护说明

- 本仓库由维护者不定期更新，新增或修改页面会通过常规提交记录。
- 欢迎提交 Issue 或 Pull Request 以改进页面内容或修复错误。
- 请勿在页面中嵌入恶意代码或违反平台政策的内容。

## 使用方式

1. 克隆或下载本仓库。
2. 直接在浏览器中打开所需的 HTML 文件。
3. 或启用 GitHub Pages 后通过对应 URL 访问。
